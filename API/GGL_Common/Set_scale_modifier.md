# Method Set_scale_modifier(modifier)

| Parameter   |  type   |              description                   |
|--           |       --|--                                          |
|   modifier      | real  |           a real between 0 and 1         |

| Returns:  | N/A                           |
|--         |                             --|

Sets a modifier that will be multiplied to "image_xscale" and "image_yscale".

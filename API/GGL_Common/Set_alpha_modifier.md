# Function Set_alpha_modifier(modifier)

| Parameter   |  type   |              description                   |
|--           |       --|--                                          |
|   modifier      | real  |           a real between 0 and 1         |

| Returns:  | N/A                           |
|--         |                             --|

Sets a modifier that will be multiplied to "image_alpha" to get the final value of "image_alpha".

# Function Scale_to(value, speed)

| Parameter   |  type   |              description                   |
|--           |       --|--                                          |
|   value      | vector2  |     the target x_scale and y_scale    |
|   speed      | real  |     the speed of the interpolation    |

| Returns:  | N/A |
|--         |                             --|

Interpolates "image_xscale" and  "image_yscale".
